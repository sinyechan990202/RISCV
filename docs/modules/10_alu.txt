[ 역할 ]
────────────────────────────────────────────────────────────────────────────────
Arithmetic Logic Unit. 두 피연산자 a_i, b_i를 받아
alu_op_i에 지정된 연산을 수행하고 result_o를 출력하는 순수 조합 논리 모듈.
레지스터 없음 — always_comb 전용.

주소 계산(LOAD/STORE/JALR), 산술 연산(ADD/SUB), 논리 연산(AND/OR/XOR),
시프트(SLL/SRL/SRA), 비교(SLT/SLTU), LUI/AUIPC/CSR PASS 등 모든 연산 처리.

[ alu_op 인코딩 (5비트) ]
────────────────────────────────────────────────────────────────────────────────
  코드    | 연산    | 설명
  --------|---------|--------------------------------------------------
  00000   | ADD     | a + b (부호 없는 덧셈, 오버플로우 무시)
  00001   | SUB     | a - b
  00010   | AND     | a & b
  00011   | OR      | a | b
  00100   | XOR     | a ^ b
  00101   | SLL     | a << b[4:0]  (논리 좌시프트)
  00110   | SRL     | a >> b[4:0]  (논리 우시프트, 0 채움)
  00111   | SRA     | $signed(a) >>> b[4:0]  (산술 우시프트, 부호 확장)
  01000   | SLT     | ($signed(a) < $signed(b)) ? 1 : 0
  01001   | SLTU    | (a < b) ? 1 : 0  (부호 없는 비교)
  01010   | LUI     | b  (a 무시, imm 상위 20비트 그대로)
  01011   | AUIPC   | a + b  (= PC + U-type imm)
  01100   | PASS_A  | a  (CSR read 값을 WB로 전달)
  others  | 0       | 0 출력

[ 주요 포트 ]
────────────────────────────────────────────────────────────────────────────────
  a_i[31:0]      — 피연산자 A (rs1 또는 PC)
  b_i[31:0]      — 피연산자 B (rs2 또는 imm)
  alu_op_i[4:0]  — 연산 코드
  result_o[31:0] — 연산 결과

[ 핵심 내부 로직 ]
────────────────────────────────────────────────────────────────────────────────
  shamt = b_i[4:0]  ← 시프트 양 (하위 5비트만 사용, RV32I 스펙)

  SLL: result = a_i << shamt
  SRL: result = a_i >> shamt           (논리 시프트, 부호 확장 없음)
  SRA: result = $signed(a_i) >>> shamt (산술 시프트, 부호 비트 유지)
  SLT: result = 0 또는 1 (MSB만 1, 나머지 0)

[ 동작 예시 ]
────────────────────────────────────────────────────────────────────────────────
  예시 1) ADD: a=5, b=3 → result=8

  예시 2) SUB: a=8, b=3 → result=5

  예시 3) SLL: a=5(0b101), b=3 → result=40(0b101000)
    shamt=3, 5 << 3 = 40

  예시 4) SRL: a=40(0b101000), b=3 → result=5(0b000101)
    논리 우시프트: 부호 상관없이 0 채움

  예시 5) SRA: a=0xFFFFFFFF(-1), b=3 → result=0xFFFFFFFF(-1)
    산술 우시프트: MSB(1)로 채워짐 → 여전히 -1

  예시 6) SLT: a=3(x2), b=5(x1) → result=1  (3 < 5이므로)

  예시 7) LUI: a=무관, b=0x00001000 → result=0x00001000
    a_i 무시, b_i(U-type imm)를 그대로 출력

  예시 8) AUIPC: a=PC=0x54, b=0x00000000 → result=0x54

  예시 9) PASS_A: a=CSR_rdata, b=무관 → result=CSR_rdata
